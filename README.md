# *ExcelViewer (Only support English Data)*

> Simple `Excel file viewer` with `React`

> **Demo site**  [https://kdilot.github.io/excelviewer/](https://kdilot.github.io/excelviewer/)
    

## Stack

- React

- Redux

- Redux Actions

- Redux Saga

- Handsontable

- Immer

- React Dropzone

- React-json-to-csv

- React-excel-file

- SASS
